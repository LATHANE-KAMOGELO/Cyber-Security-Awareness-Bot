# Cybersecurity Awareness Bot (Part 1)

## Description
The Cybersecurity Awareness Bot is a console-based chatbot developed in C#. The purpose of this chatbot is to educate users about cybersecurity topics such as password safety, phishing attacks, safe browsing, malware, and online scams.

The chatbot interacts with the user through a simulated conversation in the console and provides useful cybersecurity tips.

This project was developed as Part 1 of the Portfolio of Evidence (POE) for the Programming module.

---

## Features

- ASCII Art title screen
- Voice greeting when the program starts
- User name input and personalised interaction
- Chatbot responses for cybersecurity topics
- Keyword detection inside sentences
- Input validation
- Typing effect to simulate conversation
- Coloured console interface
- Exit commands (`exit`, `quit`, `bye`)
- Structured code using multiple classes

## How to Run the Program

1. Open the project in Visual Studio.
2. Build the project.
3. Run the program.
4. Enter your name when prompted.
5. Ask the chatbot cybersecurity-related questions.

Example questions:
- How are you?
- What is your purpose?
- Tell me about passwords
- What is phishing?
- What is malware?
- What are scams?
