## Project Description
The Cybersecurity Awareness Bot is a C# console-based chatbot designed to educate users about important cybersecurity topics such as password safety, phishing, malware, scams, safe browsing, and general cybersecurity awareness.

The chatbot simulates a conversation with the user, allowing them to ask questions and receive helpful responses. It also includes visual and interactive features such as ASCII art, coloured text, a typing effect, and a voice greeting to enhance user experience.

---

## Features

- ASCII Art title screen displayed at startup
- Voice greeting using a WAV audio file
- User name input and personalised interaction
- Chatbot responses based on keyword detection
- Input validation for empty or invalid input
- Coloured console interface for better readability
- Typing effect to simulate real conversation
- Exit commands (`exit`, `quit`, `bye`)
- Structured code using multiple classes

---

## Cybersecurity Topics Covered

The chatbot can answer questions about:

- Password safety

## File Structure

- Program.cs → Starts the application

- greeting.wav → Audio file for greeting

## How to Run the Program

1. Open the project in Visual Studio
2. Ensure `greeting.wav` is inside the project
3. Build the project
4. Run the application
5. Enter your name
6. Ask cybersecurity questions

To exit the program, type:
exit
quit
bye
